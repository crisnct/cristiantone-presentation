Jobs Hunter

The app that hunts jobs for you using ChatGPT API and pings you on WhatsApp when it finds a good one.

Project Description

Java 25 + Spring Boot 4 service that keeps a user profile (prompt, CV, phone, time interval), searches for new jobs with ChatGPT, and can push results to WhatsApp via Twilio.
Currently working on project together with another dev Andrei Lazar and here you can see our scheduled tasks: Jobshunter board on Trello

Technical Stack

Java 25
Spring Boot 4
MySQL
Liquibase
ChatGPT 5.1 API
Twillio API
Hibernate/Ehcache
Maven
