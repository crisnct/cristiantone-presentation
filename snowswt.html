SnowSWT

A Desktop Snowing Simulator and Game Demo

Project Description

SnowSWT is a desktop application developed with the Standard Widget Toolkit (SWT), serving as both a charming snowing simulator and a small game demo. This project showcases proficiency in Java desktop application development, particularly in creating interactive graphical user interfaces and handling basic game logic. It highlights the ability to leverage SWT for rich client applications, demonstrating skills in graphical rendering and event handling within a desktop environment.

Technical Stack

  • JDK 22: Development environment..
  • Standard Widget Toolkit (SWT): Used for building the desktop graphical user interface and rendering.
  • Github Actions: For CI/CD pipeline.
  • Git: Version control system for code management.
  • Maven: Build automation tool for project management and dependency handling.
