Metal Investment

Real-time Profit Notifications for Precious Metals

Project Description

The Metal Investment project is a practical desktop tool designed to empower users with real-time insights into potential profits from selling precious metals (gold, silver, platinum) on the Revolut platform. It continuously monitors market data and notifies users when favorable selling conditions are met, helping them make informed decisions to maximize their returns. This application demonstrates a robust approach to data retrieval, real-time updates, and effective user notification systems, providing a tangible benefit to investors. The project is monitoring the precious metals price at Bloomberg and Galmarley, and it's using some formula to approximate the metal price at Revolut.

Technical Stack

  • Java 22
  • Spring Boot: Framework for backend of a webapp with typical starters: web, data‑JPA, security, mail, actuator.
  • Liquibase: Library for tracking, managing and applying database schema changes.
  • Resilience4j: Lightweight fault tolerance library that implements resilience patterns, like circuit breakers, rate limiters, and retries, to make applications more robust and able to handle failures gracefully
  • Unirest: HTTP client library for making API calls to fetch real-time metal prices.
  • External APIs: For fetching real-time precious metal prices.
  • Hibernate/Ehcache: caching integration for database
  • MySQL: database
  • Maven: Build automation tool for project management and dependency handling.
  • Git: Version control system for collaborative development and code management.
