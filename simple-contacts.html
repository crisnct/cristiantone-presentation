Simple Contacts Manager

Address book

Project Description

The Simple Contacts Manager it's a Spring Boot address book with login, contact CRUD, Kafka sign-up events, MySQL persistence, and a Dockerized weather microservice companion.

Technical Stack

  • Java 21
  • Spring Boot 3
  • Kafka
  • Docker
  • MySQL
  • Liquibase
  • External APIs
  • Hibernate/Ehcache
  • Maven
  • Git
